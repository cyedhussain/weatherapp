# 🌤️ Weather App (Flutter + Provider)

A modern and clean **Weather Application built with Flutter** using the **Provider state management**.  
It provides real-time weather updates, search functionality, and multi-day forecasts with a beautiful UI.

---

## 📱 Features

- 🌍 Search weather by city name
- 🌡️ Real-time current weather data
- 🌦️ Weather condition with icons
- 🌬️ Wind speed, humidity, clouds, and feels-like temperature
- 📅 3-day / forecast screen
- ⏱️ Hourly weather updates
- 🎨 Clean modern UI (light theme)
- ⚡ Fast and responsive design
- 🧠 State management using Provider

---

## 🛠️ Tech Stack

- Flutter (UI Framework)
- Dart
- Provider (State Management)
- REST API (Weather API)
